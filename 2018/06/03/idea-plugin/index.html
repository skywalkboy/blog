自己在用的IDEA插件

Jun 3, 2018

周末用了SpaceSnif清理了下电脑WINDOW，看到升级之前IDEA的版本占用了几个G内存，
就直接删除了。后面就不得不重新安装IDEA上面的插件跟设置了，之前安装的插件不多，常用的也就那么几个，记录下下次备用

  1. Mybatis plugin by codemagic
  2. Material Theme UI
  3. Grep console
Mybatis plugin

Mybatis plugin是mybatis开发的得力，需要crack。mybatis plugin crack这个博主的版本可以使用，但是插件的版本跟IDEA的版本关联比较大，ideaagent v1.2版本对应的插件版本是3.58，需要自己从网上下载安装。

Material Theme UI

顾名思义就是一个UI插件，安装后就是程序员特色的颜色效果

Grep console

IDEA控制台日志输出的时候不同日志级别可以用颜色自定义区分

总结

继续发现好用的插件，其实我们也可以通过浏览插件库查看比较常用的插件。
